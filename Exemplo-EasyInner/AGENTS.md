# AGENTS.md — Arquivos de exemplo temporários

## Escopo
Este diretório contém apenas **artefatos de referência** que demonstram comportamentos do sistema legado. Nenhum arquivo aqui deve ser consumido diretamente pelo aplicativo principal.

## Diretrizes obrigatórias
- **Não modifique** nenhum arquivo dentro desta pasta.
- **Não referencie** nem invoque esses arquivos a partir do código produtivo; trate-os somente como material de consulta.
- **Não copie** a organização, nomes de tipos ou estrutura do diretório para o projeto definitivo, exceto quando for explicitamente solicitado reaproveitar um método ou comportamento específico.
- Assim que as funcionalidades forem transpostas para o novo código, este diretório será removido. Use-o apenas para inspeção temporária.

## Uso permitido
- Ler o código para entender fluxos, assinaturas ou protocolos que precisem ser mimetizados.
- Extrair trechos conceituais (ex.: lógica de parsing) e reimplementá-los no projeto oficial seguindo as convenções vigentes.
